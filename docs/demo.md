# Demo script

## Demo promise

Show the whole loop in under two minutes:

1. a local Mac task arrives
2. prior experience is retrieved
3. multiple proposal lanes compete
4. deterministic evaluation selects a winner
5. experience is written back
6. a later task benefits from the new memory
7. the final task bridges into an H200 lane

## Narration

1. "This system is organizing validated research experience, not chat transcripts."
2. "The local Mac loop uses deterministic evaluation so keep and discard decisions stay inspectable."
3. "Replay-synthesizer wins once prior experience becomes useful, which is the flywheel."
4. "The H200 lane is modeled as a handoff bundle, not a separate product."

local search on macOS can produce reusable, scored experience that later conditions cluster-scale training decisions.
